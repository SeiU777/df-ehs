第一章：專案簡介與系統架構

1.1 這個系統是什麼？

大豐環保法務及環安課的職業災害分析報告系統，功能包括：

  • 職災統計報告網站（GitHub Pages 自動部署）
  • 個案職災分析報告產生器（AI 輔助，產出 .docx）
  • 安委會季度簡報生成工具
  • 教育訓練教材網站

1.2 系統運作流程

┌─────────────────┐ ┌──────────────────┐ ┌─────────────────┐ │ Excel 職災資料 │────→│ Claude Code │────→│ HTML 報告檔案 │ │ （每月更新） │ │ （AI 讀取分析） │ │ （df-ehs/docs/） │ └─────────────────┘ └──────────────────┘ └────────┬────────┘ │ git push ▼ ┌─────────────────┐ │ GitHub Pages │ │ （自動部署網站） │ └─────────────────┘ │ ▼ https://seiu777.github.io/df-ehs/

1.3 四大組件

組件技術用途
A. 靜態網站HTML + CSS + JS職災報告網站，部署在 GitHub Pages
B. Python 腳本Python 3.12+讀取 Excel、處理 PDF、資料分析
C. 簡報工具Node.js生成安委會 PowerPoint / Word
D. Claude CodeAI 助手自動更新報告、分析職災案件

第三章：環境安裝

3.1 快速安裝（已有 Git / Python / Node.js）

最快的方式
  1. 複製整個資料夾到 C:\Users\{你的名稱}\Documents\claude\職安\
  2. 雙擊執行 setup.bat
  3. 完成！

3.2 從零開始安裝

1

安裝 Git

前往 https://git-scm.com/download/win 下載，安裝時保持預設。

驗證：git --version

2

安裝 Python

前往 https://www.python.org/downloads/ 下載 3.12 以上版本。

重要
安裝時務必勾選底部的 "Add Python to PATH"，否則命令列找不到 Python。

驗證：python --version

3

安裝 Node.js

前往 https://nodejs.org/ 下載 LTS 版本，安裝時保持預設。

驗證：node --version

4

安裝 VS Code（建議）

前往 https://code.visualstudio.com/ 下載。

建議安裝擴充功能：Claude Code（Anthropic 官方）

5

放置專案

將整個資料夾複製到：

C:\Users\{你的使用者名稱}\Documents\claude\職安\

建議使用相同路徑，可減少設定修改。

6

執行安裝腳本

雙擊 setup.bat，自動安裝所有依賴和 Claude Code 設定。

7

測試網站

cd df-ehs\docs
python -m http.server 3300

瀏覽器開啟 http://localhost:3300，看到首頁即成功。

第五章：Claude Code 設定

5.1 什麼是 Claude Code？

Anthropic 出品的 AI 編程助手，能直接操作電腦上的檔案。在本專案負責：

  • 讀取 Excel → 自動更新 HTML 報告
  • 分析事故截圖 → 產出 .docx 分析報告
  • 輔助安委會簡報製作
非必要工具
沒有 Claude Code 也可以手動編輯 HTML 報告。但有了它，每月更新報告只需要一句話。

5.2 修改 Shell 路徑（重要！）

開啟 ~/.claude/settings.json，將 SHELL 路徑改成你電腦的 Git Bash：

{
  "env": {
    "SHELL": "C:\\Users\\{你的使用者名稱}\\AppData\\Local\\Programs\\Git\\bin\\bash.exe"
  }
}

setup.bat 已自動複製 3 個職安專用 Skills：

  • ~/.claude/skills/7-accident-analysis/report-template.js — 第 9 行 modulePath

第六章：日常操作流程

6.1 月度報告更新

最常用的流程：每月更新職災數據後重新生成報告。

1

取得最新 Excel

放到 參考資料/公司職災資料/ 目錄

2

開啟 Claude Code

輸入：「資料更新了，幫我更新報告」

3

部署到 GitHub Pages

cd df-ehs
git add -A
git commit -m "更新 YYYY-MM 月報"
git push

等 1-2 分鐘，前往 https://seiu777.github.io/df-ehs/ 確認。

6.2 個案職災分析

1

準備事故截圖

發生單位表單截圖 + 工安單位表單截圖（如有）

2

請 Claude 分析

輸入：「這是公司新的職災案件，請你幫我做分析報告」，然後提供截圖。

3

取得報告

Claude 會產出 .docx 報告，存放在 參考資料/公司職災資料/分析報告/{年份}/

6.3 本地預覽網站

cd df-ehs\docs
python -m http.server 3300

瀏覽器開啟 http://localhost:3300，按 Ctrl+C 結束。

第七章：常見問題排解

Q: python 指令找不到？

Windows 上有時需要用 py 代替 python：

py --version
py -m http.server 3300

或重新安裝 Python 並勾選 "Add Python to PATH"。

Q: git push 被拒絕？

你需要 seiu777/df-ehs repo 的存取權限。

請聯繫 repo 管理員，將你的 GitHub 帳號加為 collaborator。

Q: 網站更新後看不到變化？

1. GitHub Pages 有 1-2 分鐘部署延遲

2. 清除瀏覽器快取：按 Ctrl+Shift+R（強制重新載入）

Q: 中文資料夾名稱有問題？ 
git config --global core.quotepath false
Q: npm install 失敗？

如果公司有代理伺服器：

npm config set proxy http://proxy.company.com:8080
npm config set https-proxy http://proxy.company.com:8080
Q: Claude Code 找不到檔案？

確認 參考資料/ 資料夾已正確放置，且路徑與 Skills 中的設定一致。

Q: Skills 沒有出現？

確認 SKILL.md 放在 ~/.claude/skills/{skill-name}/SKILL.md。

資料夾名稱必須與 Skill 名稱完全一致。

Q: setup.bat 閃退？

右鍵 setup.bat → 「以系統管理員身分執行」。
