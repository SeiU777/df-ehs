第一章：網站總覽

1.1 網站是什麼？

大豐環保法務及環安課的職業災害分析報告網站，部署在 GitHub Pages 上，所有員工可透過網址瀏覽。

1.3 技術特色

  • 純靜態網站：HTML + CSS + JavaScript，無後端伺服器
  • 密碼保護：XOR 加密驗證，sessionStorage 保持登入
  • 深色模式：支援淺色/深色主題切換，跟隨系統設定
  • 響應式設計：手機、平板、電腦三種螢幕尺寸
  • Chart.js 圖表：互動式統計圖表（折線圖、圓餅圖、長條圖）
  • 側邊目錄：可收合的章節導航（桌面常駐、行動裝置隱藏）

2.2 登入方式

1

  • 狀態儲存在 sessionStorage（關鍵：df-ehs-auth）
  • 關閉瀏覽器或關閉分頁後會自動登出，下次需重新輸入

2.5 技術細節

密碼驗證邏輯在 js/auth.js 中：

  • 加密方式：XOR（金鑰 = 42）
  • 編碼後資料：[110,75,26,25,26,24,76,69,68,24,26,24,28,15]
  • 驗證函式：逐字元比對 input.charCodeAt(i) ^ 42 === data[i]
  • 非高安全加密，僅防止一般瀏覽，有技術背景的人可解碼

2.6 如何修改密碼

1

開啟 js/auth.js

2

決定新密碼（例如 NewPass2026!）

3

用以下 JavaScript 計算新的編碼資料：

const password = 'NewPass2026!';
const key = 42;
const encoded = Array.from(password).map(c => c.charCodeAt(0) ^ key);
console.log(encoded); // 貼入 auth.js 的 _d 陣列
4

將結果替換 _d 陣列

5

提交並推送

4.2 報告結構（每年相同）

每份年度報告包含以下章節：

  1. 封面 — 年度標題、資料期間、公司名稱
  2. KPI 總覽 — 4 個指標卡片（總案件數 / 損失工日 / 嚴重事故 / 月均件數）
  3. 事故分類統計 — 工安 / 交通 / 虛驚的件數與佔比（圓餅圖）
  4. 事故類型深度分析 — 各類型（被夾被捲、跌倒、墜落等）的件數與嚴重度
  5. 月份趨勢 — 每月事故件數折線圖 / 長條圖
  6. 部門分析 — 各部門事故頻率
  7. 原因分析 — 不安全行為 / 不安全狀況 / 基本原因統計
  8. 嚴重事故案例 — 重大案件詳細分析
  9. 改善建議 — 工程 / 管理 / 教育 / PPE 四層次
  10. 結語 — 年度總結與展望

4.3 使用的視覺元件

元件CSS Class用途
KPI 卡片.kpi-grid + .kpi-card指標數字展示
圓餅圖CSS conic-gradient比例分佈圖（純 CSS，無外部套件）
長條圖CSS width 百分比頻率 / 嚴重度比較
Chart.js 圖表<canvas>互動式折線圖、長條圖
重點卡片.highlight-card綠色邊框，正面指標
警示卡片.warning-card橙色邊框，需注意項目
危險卡片.danger-card紅色邊框，嚴重警示
徽章.badge-danger/warning/success嚴重度標籤
側邊目錄.sidebar + .toc-chapter章節導航（可收合）

  • 顯示在導航列右側

  • 每個課程章節結束後有測驗題

    • 未達 80 分顯示「請複習後重新作答」（紅色）
    • 提交後不可更改
  • CSS Class：.quiz-section、.quiz-question、.quiz-option、.quiz-submit-btn、.quiz-result

法規引用

  • 課程中引用的法規條文用特殊樣式標示

第十三章：JavaScript 模組

13.1 auth.js — 密碼驗證

公開 API（window.dfAuth）：

方法說明
dfAuth.verify(input)驗證輸入的密碼是否正確，回傳 boolean
dfAuth.isAuthenticated()檢查是否已登入（sessionStorage）
dfAuth.setAuthenticated()設定已登入狀態
dfAuth.logout()登出（清除 sessionStorage）

  • 頁面載入時讀取 localStorage.theme 或系統偏好
  • 動態注入切換按鈕到 <body>
  • 監聽系統主題變更事件

13.3 sidebar.js — 側邊目錄

函式說明
toggleSidebar()切換側邊欄開/關
closeSidebar()關閉側邊欄（含動畫）

第十四章：如何修改與更新

14.1 更新報告數據

1

取得最新職災 Excel

2

用 Claude Code 說「資料更新了，幫我更新報告」

3

Claude 讀取 Excel → 更新 HTML 中的數據、圖表、KPI

4

git add → git commit → git push

14.2 新增年度報告

1

複製上一年的 HTML（如 2025.html → 2026.html）

2

修改：標題、封面顏色（每年不同色系）、sidebar 目錄

3

替換所有數據和圖表

4

在 index.html 新增年度卡片

5

在 overview.html 加入新年度數據

6

建立對應的摘要版 2026-summary.html

14.3 新增安委會季報

1

複製 committee/2026-Q1.html → committee/2026-Q2.html

2

修改內容（法定 12 項各項更新）

3

在 committee/index.html 啟用 Q2 卡片（移除 disabled）

4

將素材照片放到 committee/images/2026-Q2/

14.4 修改教育訓練

  • 編輯 training/hq.html 或 training/quanxing.html
  • 測驗題的正確答案在 .quiz-question 的 data-answer 屬性中

14.5 修改密碼

見 第二章 2.6 節

14.6 修改樣式

  • 改品牌色 / 字型 → 編輯 css/common.css 的 :root 變數
  • 改深色模式 → 編輯 css/dark.css
  • 改側邊欄 → 編輯 css/sidebar.css
  • 改教育訓練外觀 → 編輯 css/training.css
