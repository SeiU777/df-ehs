📷 圖片上傳工具

直接從網頁上傳照片到 GitHub，本機執行 git pull 即可同步
📄 使用說明
  1. 首次使用：在下方輸入 GitHub Personal Access Token（需 Contents: Write 權限），儲存後長期有效。
  2. 選擇季別：選擇要上傳到哪一季的資料夾（如 2026-Q1）。
  3. 拖入照片：拖曳或點選檔案，可同時上傳多張。
  4. 確認檔名：檢查或修改存檔名稱（英文命名，如 nurse-service.jpg）。
  5. 上傳：點擊「開始上傳」，照片會直接 commit 到 GitHub repo。
  6. 本機同步：在 VS Code 終端機執行 git pull，即可下載到本機資料夾。

🔑 GitHub Token 設定

☐ 尚未設定

申請方式：GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → 選擇 Repository: df-ehs → Contents: Read and Write
