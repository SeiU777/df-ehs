# 參考資料取得與放置指南

> 參考資料（1.1GB+）不在 Git 中，需要另外取得和放置。

---

## 為什麼參考資料不在 Git 裡

1. **檔案太大**：參考資料超過 1.1GB（主要是 PDF 和圖片），GitHub 有容量限制
2. **含員工個資**：職災 Excel 包含員工姓名、身分證字號等敏感資料
3. **二進位檔案**：Excel/PDF/圖片不適合 Git 版本控制

---

## 取得方式

以下任一方式取得參考資料：

1. **向法務及環安課負責人索取** — 取得最新版本
2. **從前任的電腦直接複製** — 複製整個 `參考資料/` 資料夾
3. **公司共用資料夾** — 如有存放在 NAS 或 OneDrive，從中下載

---

## 放置位置

將 `參考資料/` 資料夾放在專案根目錄下（與 `df-ehs/` 同層）：

```
職安/
├── df-ehs/          ← Git repo（已 clone）
├── 參考資料/         ← 放在這裡
│   ├── 公司職災資料/
│   ├── 安委會/
│   ├── 職災分析方法/
│   ├── 教育訓練/
│   └── 原google分析內容與簡報/
└── *.py             ← Python 腳本（也需從前任複製）
```

| 檔案/資料夾 | 用途 | 重要性 |

### 安委會/ — 簡報工具與素材

| 檔案/資料夾 | 用途 | 重要性 |

## 資訊安全提醒

- 職災 Excel 包含**員工個人資料**（姓名、身分證字號、出生年月日等）
- **絕對不要**上傳到 GitHub 或其他公開平台
- **絕對不要**透過未加密的管道傳送
- 離開電腦時請鎖定螢幕
- 定期從公司系統取得最新數據，保持資料同步
